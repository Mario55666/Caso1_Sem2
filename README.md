Semiótica de la Imagen — Simulador Interactivo Nike
Este proyecto es una herramienta pedagógica interactiva diseñada para estudiantes de Diseño Gráfico y Ciencias de la Comunicación. A través del estudio de caso de Nike ("Just Do It" y la campaña "So Win" 2025), el simulador permite explorar de forma práctica y visual los conceptos fundamentales de la semiótica contemporánea.

🎯 Objetivo de Aprendizaje
Desarrollar el pensamiento crítico sobre la naturaleza mediatizada de la comunicación visual, comprendiendo cómo los sistemas de signos (color, tipografía, composición e imagen) construyen significados que exceden la realidad física del producto.

🚀 Características Principales
Marco Teórico Integrado: Tarjetas interactivas con las posturas de Saussure, Peirce, Eco y Blanco & Bueno.

Comparativa Denotación vs. Connotación: Sistema de hotspots interactivos sobre piezas gráficas de Nike que revelan análisis técnicos y simbólicos.

Visualización de Datos: Gráficos de barras desarrollados con CSS puro para comparar la eficacia comunicativa de las estrategias.

Evaluación Formativa: Cuestionario de 5 preguntas con retroalimentación inmediata y cálculo de puntaje en tiempo real.

Reflexión Crítica: Formulario de envío integrado con validación de datos para la entrega de tareas vía correo electrónico.

Diseño Adaptativo (Responsive): Experiencia optimizada para escritorio, tablets y móviles.

Optimización de Impresión: Hoja de estilos específica para generar reportes en PDF limpios y profesionales.

🛠️ Tecnologías Utilizadas
HTML5: Estructura semántica.

CSS3: Diseño avanzado mediante CSS Grid, Flexbox, animaciones personalizadas y variables nativas.

Vanilla JavaScript: Lógica de interacción, uso de IntersectionObserver para animaciones al hacer scroll y validación de formularios.

Google Fonts: Tipografías Playfair Display (serif para jerarquía académica) y DM Sans (sans-serif para lectura técnica).

📖 Contenido Académico Incluido
El simulador aborda la transición del signo desde cuatro perspectivas:

Ferdinand de Saussure: El signo en la vida social y el contrato colectivo.

Charles S. Peirce: La tríada del signo (Ícono, Índice y Símbolo).

Umberto Eco: La semiosis total y el signo como sistema de códigos activados.

Blanco y Bueno: El análisis de sistemas de significación complejos (raza, género, poder).

💻 Instalación y Uso
Descarga el archivo index.html.

Ábrelo en cualquier navegador moderno (Chrome, Firefox, Edge, Safari).

No requiere dependencias externas ni servidores, funciona de forma local.
